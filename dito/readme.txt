Boost your site's signups by adding a top bar with social login and send segmented notifications to your users.

== Description ==

This plugin automates your integration with the Dito platform (http://dito.com.br).
It automatically inserts the Dito external code and the installation and tracking scripts in your wordpress site.
You can activate a top bar that uses the Social Login, a plugin made by Dito that enable you to login in your site or blog with Facebook and Google.
You will need a Dito account to user this plugin properly.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
